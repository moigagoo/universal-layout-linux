# Универсальная английская и русская раскладка для Linux

## Установка

Замените файлы `/usr/share/X11/xkb/symbols/us`, `/usr/share/X11/xkb/symbols/ru` и `/usr/share/X11/xkb/rules/evdev.xml` файлами из этого репозитория.
